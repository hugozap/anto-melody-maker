# Anto Melody Maker

A grid-based melody composer for Anto. Click cells to place notes on a 16-step timeline, then press Play to hear your melody.

Built with vanilla HTML, CSS, and JavaScript. Uses the Web Audio API for sound synthesis and Canvas 2D for visual effects.

Sister project to [Anto Drum Box](https://github.com/hugozap/anto-drum-box).

## How to Run

```bash
npx serve . -l 3335
```

Then open http://localhost:3335

## How to Test

```bash
npx playwright test --reporter=list
```
